# go-consul

封装了 Consul HTTP Api 的SDK

Consul的文档👁这里: [consul](https://www.consul.io/api-docs)

## 目前完成

- check
    - 注册
    - 注销

- KV Store
    - 新建/更新指定key
    - 读取指定key
    - 删除指定key
